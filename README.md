# Global Social Media Analytics Dashboard

This repository now includes a simple static analytics dashboard demo built with HTML, CSS and Chart.js and ready to be served with GitHub Pages.

How to publish with GitHub Pages

1. In the repository, go to Settings â†’ Pages.
2. Under Source choose the branch `main` and the folder `/ (root)` and click Save.
3. After a minute the site will be available at: `https://comet221.github.io/Global-Social-Media-Analytics-Dashboard/`

How to edit

- Edit `index.html` and `assets/style.css` directly in the repository and push changes.
- Replace the sample `data` object in the inline script with your real analytics JSON export or wire it to an API.
